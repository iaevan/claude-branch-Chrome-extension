# Contributing to Claude Branch Chat

First off, thanks for taking the time to contribute! This project helps me manage my AI workflows, and I appreciate any help making it better.

## How Can I Contribute?

### Reporting Bugs
* Check the **Issues** tab to see if the bug has already been reported.
* If not, open a new issue. Include your Chrome version and a screenshot of the console if there are errors.

### Suggesting Enhancements
* Open an issue with the tag `enhancement`.
* Explain why the feature would be useful and how it should work.

### Pull Requests (PRs)
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add examples.
3. Ensure the `manifest.json` version is incremented if applicable.
4. Write a clear PR description of what you changed.

## Style Guide
* Keep Javascript "Vanilla" (no heavy libraries).
* Maintain the minimalist, orange-themed CSS.
* Ensure all paths in `manifest.json` remain correct.

## Privacy Policy
Any contribution must respect user privacy. No code that tracks user data or sends information to external servers will be accepted.
